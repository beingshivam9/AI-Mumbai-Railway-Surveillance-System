# AI Railway Surveillance System

This project detects unattended bags in railway stations using computer vision.

The system analyzes CCTV feeds and identifies objects that remain unattended for a certain duration. Once detected, the system triggers an alert to help security personnel respond quickly.

This project was developed as a B.Tech Telecommunications final year project.

Future scope:
- Real-time CCTV monitoring
- Integration with railway security systems
- Automatic alert generation
